# Controle de Despesas

Um aplicativo web simples para controlar despesas compartilhadas entre duas pessoas. Permite adicionar despesas, visualizar o total mensal e ver a contribuição de cada um em porcentagem.

## Funcionalidades

- Adicionar despesas com valor, quem pagou, descrição e data
- Visualizar resumo mensal com totais e porcentagens
- Lista de todas as despesas do mês atual
- Dados salvos localmente no navegador

## Como Usar

1. Abra o aplicativo no navegador
2. Adicione despesas preenchendo o formulário
3. Veja o resumo e a lista de despesas do mês

## Desenvolvimento

Este é um projeto [Next.js](https://nextjs.org) criado com [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

### Executar o servidor de desenvolvimento

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador para ver o resultado.

### Build para produção

```bash
npm run build
npm start
```
